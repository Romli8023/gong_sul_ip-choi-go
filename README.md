🤖 Lego EV3를 이용한 능동형 자동 수평 컵홀더 (Active Self-Balancing Cup Holder)
📝 프로젝트 요약 (Summary)
본 프로젝트는 Lego EV3와 자이로 센서, 4개의 모터를 활용하여 외부의 흔들림이나 기울어짐에도 스스로 수평을 유지하는 '능동형 컵홀더'를 개발하는 것을 목표로 합니다.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

🎯 개발 동기 (Motivation)
휠체어나 차량과 같은 이동수단에 장착된 기존 컵홀더는 경사로나 충격에 매우 취약하여 내용물이 쏟아지기 쉽습니다. 저희는 이러한 불편함과 위험을 해결하고자, 어떤 상황에서도 사용자가 안전하고 편리하게 음료를 보관할 수 있는 '쏟아지지 않는' 스마트 컵홀더를 개발하게 되었습니다.


-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

✨ 주요 기능 (Key Features)
능동적 수평 유지 시스템 (Active Self-Balancing System)

실시간 기울기 감지: 자이로 센서(IMU)가 컵홀더의 미세한 기울기 변화를 실시간으로 감지합니다.



지능형 자세 제어: 감지된 데이터를 바탕으로 PID 제어 알고리즘이 즉각적으로 모터를 제어할 최적의 값을 계산합니다.


신속한 수평 복원: 4개의 모터가 계산된 값에 따라 정밀하게 움직여, 경사로나 갑작스러운 흔들림에도 컵홀더의 수평을 완벽하게 복원합니다.


-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

🚀 기대 효과 (Expected Effects)
안정성 증대: 휠체어 사용자나 운전자 등이 이동 중에도 음료를 쏟을 걱정 없이 안전하게 보관할 수 있습니다.


사용자 편의성 향상: 기존 컵홀더처럼 수동으로 각도를 조절할 필요가 없어 매우 편리합니다.


기술적 응용: 본 프로젝트의 짐벌(Gimbal) 기술은 카메라, 스마트폰 거치대 등 흔들림 방지가 필요한 다양한 분야에 확대 적용될 수 있습니다.


-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

🛠️ 기술 설명 (Tech Stack)
하드웨어 (Hardware)
Main Controller: Lego EV3 Intelligent Brick

Sensor: Lego EV3 Gyro Sensor (IMU)

Actuator: Lego EV3 Large Motor x 4

소프트웨어 (Software)
Language: Python

Library: ev3dev2

Core Algorithm: PID 제어 (Proportional-Integral-Derivative Control)

기울어진 각도(Error)를 바탕으로, 오차의 비율(P), 누적값(I), 변화율(D)을 모두 고려하여 모터를 정밀하게 제어하는 알고리즘입니다. 이를 통해 목표 각도(수평)로 빠르고 부드럽게 복귀할 수 있습니다.

전체 제어 코드는 아래 링크에서 확인하실 수 있습니다.
